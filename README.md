# Advanced Student Management System (Python)

A simple **Object-Oriented Python project** that demonstrates how a student management system can be built using classes, objects, and Python magic methods.

This project was created as part of my learning journey as a **First-year Computer Science Engineering student**, while exploring Python *OOP* concepts and practical program design.

-----


## Project Overview

The program manages student records and performs operations such as:

- Adding students
- Displaying all students
- Ranking students based on CGPA
- Searching students using roll numbers
- Showing the top N students
- Calculating the class average CGPA
- Demonstrating Python *magic methods*

The goal of this project was to understand how **real-world systems can be modeled using *classes and objects***.

-----

## Concepts Used

This project demonstrates the use of:

- Python Classes and Objects
- Constructors(`__init__`)
- Magic methods:
  - `__str__`
  - `__repr__`
  - `__lt__`
  - `__eq__`
  - `__add__`
  - `__len__`
- Sorting Objects
- List Management
- Basic System Design
- Modular Programminng

-----

## Features

- Add multiple students to the system
- Display student details in a readable format
- Rank students automatically based on CGPA
- Search students using roll numbers
- View the **Top N students**
- Calculate the **average CGPA of the class**
- Demonstrates custom operator behavior using magic mathods
